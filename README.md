# WordPress Dockerized
Simple docker solution for starting quickly with WordPress.

### Installation
```
git clone https://github.com/gmaccario/wordpress-dockerized.git wordpress-dockerized
cd wordpress-dockerized
docker-compose up -d
docker exec -it  wp-lemp-wordpress /bin/bash
```

### Urls
* WP Frontend: http://localhost
* WP Backend: http://localhost/wp-admin
* PhpMyAdmin: http://localhost:1234
